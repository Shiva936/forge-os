# `.claude/memory/` — non-canonical notes only

Use this directory for **optional** material that helps humans or agents **navigate** the repo:

* local environment quirks
* service URLs for manual verification
* personal checklists

## Hard rules

* Do **not** store **release** or **plan** truth here.
* Do **not** duplicate **`/.forge/releases/changelog.json`** history.
* Do **not** replace **`.claude/CLAUDE.md`** or **`.cursor/rules/*.mdc`** with “shadow policies” stored here.

If a note becomes normative, promote it to **`.cursor/rules/`** (law), **`.forge/scripts/`** (automation), or **`.claude/CLAUDE.md`** (charter)—not an unreviewed memory file.
